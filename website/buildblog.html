    # 博客设置

    博客下载好并不能立刻执行

    先打开blog-demo（如果你没改名字的话就是这个默认名）这个文件夹

    修复好可以测试，输入npm run dev

    等待好它会给你一个网址http://localhost:8080/默认是这个

    打开这个网址就可以查看你的博客主页是什么样的

    准备好了就得到比较困难的一步（耗时较多）配置js文件，也就是配置博客设置

    # GitHub博客上传

    确保GitHub注册完毕和git上传工具

    在GitHub上新建一个库,由于要确保网站可用

    请确保下面的图片

    然后库名字就是（用户名.github.io）

    如图所示

    # 编译

    在blog-demo目录下

    还是shift和鼠标右键打开命令行

    使用npm run build编译

    弄好后打开目录，会出现编译好的文件

    在这个目录下使用3条命令

    # git init

    这是在dist目录下创建一个可管理的库，这时会创建一个.git的文件夹，如果已经有了（在dist目录下不是其他目录）就不需要再重复一遍否则会报错

    # git add .

    add跟.之间有一个空格

    这是将所有文件加入上传队列

    这时还没上传

    # git commit -m "first commit"

    上传的起名

    # 上传

    git remote add origin （github提供的ssh）连接库

    git branch -M main git push -u origin main正式上传

    好了就可以访问你的网站了，得等几分钟，因为还没部署完成，网页会是404
